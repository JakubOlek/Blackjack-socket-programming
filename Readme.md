# Blackjack - socket programming
Card game using socket programming. Server written in C and client in Java. Created as "Computer Networks" course final project at Adam Mickiewicz University. 
# Run
server directory:

>**$ gcc main.c -o server**

>**$ ./server**

client directory:
>**$ javac Main.java**

>**$ java Main**
